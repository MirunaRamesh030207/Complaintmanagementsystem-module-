# Complaint Management System

## 📌 Overview
A web-based Complaint Management System built using ASP.NET Core MVC and MySQL. The system allows users to manage complaints efficiently with complete CRUD functionality.

## 🚀 Features
- Add Complaint
- View Complaint Details
- Edit Complaint
- Delete Complaint
- Search Complaints
- Filter by Status
- Dashboard with Complaint Statistics
- Responsive Bootstrap UI

## 🛠️ Technologies Used
- ASP.NET Core MVC
- C#
- Entity Framework Core
- MySQL
- Bootstrap 5
- Chart.js
